function twoSum(nums: number[], target: number): number[] {
    for (let i = 0; i < nums.length; i++) {
        for (let j = i + 1; j < nums.length; j++) {
            if (nums[i] + nums[j] === target) {
                return [i, j];
            }
        }
    }
    return []; // This line will never be reached if the problem guarantees a solution
}

// Example 1:
// Input: nums = [2,7,11,15], target = 9
// Output: [0,1]
// Explanation: nums[0] + nums[1] = 2 + 7 = 9
console.log(twoSum([2,7,11,15], 9)); // Expected output: [0,1]

// Example 2:
// Input: nums = [3,2,4], target = 6
// Output: [1,2]
// Explanation: nums[1] + nums[2] = 2 + 4 = 6
console.log(twoSum([3,2,4], 6)); // Expected output: [1,2]

// Example 3:
// Input: nums = [3,3], target = 6
// Output: [0,1]
// Explanation: nums[0] + nums[1] = 3 + 3 = 6
console.log(twoSum([3,3], 6)); // Expected output: [0,1]
```

- The time complexity is O(n^2) because in the worst case, we need to check every pair of numbers in the array
- The space complexity is O(1) as we only use a constant amount of extra space.
</details>

function twoSum(nums: number[], target: number): number[] {
    const numMap = new Map<number, number>();

    for (let i = 0; i < nums.length; i++) {
        const complement = target - nums[i];
        if (numMap.has(complement)) {
            return [numMap.get(complement)!, i];
        }
        numMap.set(nums[i], i);
    }

    return []; // This line will never be reached if the problem guarantees a solution
}

// Example 1:
// Input: nums = [2,7,11,15], target = 9
// Output: [0,1]
// Explanation: nums[0] + nums[1] = 2 + 7 = 9
console.log(twoSum([2,7,11,15], 9)); // Expected output: [0,1]

// Example 2:
// Input: nums = [3,2,4], target = 6
// Output: [1,2]
// Explanation: nums[1] + nums[2] = 2 + 4 = 6
console.log(twoSum([3,2,4], 6)); // Expected output: [1,2]

// Example 3:
// Input: nums = [3,3], target = 6
// Output: [0,1]
// Explanation: nums[0] + nums[1] = 3 + 3 = 6
console.log(twoSum([3,3], 6)); // Expected output: [0,1]
```
- This solution has a time complexity of O(n). We iterate through the array once, performing constant-time operations for each element.
- A space complexity of O(n). The space used by the Map grows linearly with the input size, hence O(n).

  <summary>Solution</summary>
  ```ts
  function isValid(s: string): boolean {
    const stack: string[] = [];
    const bracketPairs: { [key: string]: string } = {
        ')': '(',
        '}': '{',
        ']': '['
    };

    for (const char of s) {
        if (char in bracketPairs) {
            if (stack.length === 0 || stack[stack.length - 1] !== bracketPairs[char]) {
                return false;
            }
            stack.pop();
        } else {
            stack.push(char);
        }
    }

    return stack.length === 0;
  }

  // Examples
  console.log(isValid("()")); // true
  console.log(isValid("()[]{}")); // true
  console.log(isValid("(]")); // false
  console.log(isValid("([)]")); // false
  console.log(isValid("{[]}")); // true
  ```

  - Time complexity: O(n), where n is the length of the input string
    - We iterate through each character in the input string exactly once
    - All operations inside the loop (pushing, popping, and checking the top of the stack) are O(1)
  - Space complexity: O(n) in the worst case
    - In the worst case, we might push all opening brackets onto the stack
    - This happens when the input string consists of only opening brackets or when it's a valid string with all opening brackets first, followed by all closing brackets
</details>
